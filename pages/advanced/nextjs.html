Curso de NextJS

Next.js é um framework para React. O que isso quer dizer? O React é uma biblioteca Javascript para construção de interfaces e o Next é considerado um framework pois adiciona várias funcionalidades em cima do React. Algumas funcionalidades do Next.js: renderização estática e pelo lado do servidor. Possui suporte ao Typescript e um serviço de tratamento de rotas muito interessante.

Conteúdos que serão abordados durante esse curso:

  • Diferenças entre ReactJS e NextJS
  • Rotas Dinâmicas
  • Data fetching
  • Static Site Generation (SSG)
  • Server Side Rendering (SSR)
  • Incremental Site Regeneration (ISR)
  • Client-side Rendering (CSR)
  • Next API
